# Security Policy

## Supported Versions

The latest released version on the default branch receives security updates.

## Reporting a Vulnerability

Please report security vulnerabilities privately, not via public issues.

- Preferred: open a private advisory via GitHub Security Advisories ("Report a vulnerability").
- Alternative: email security@ogulcanaydogan.com.

Include a description, reproduction steps, affected version/commit, and impact.

## Response Targets

- Acknowledgement within 48 hours.
- Initial assessment within 5 business days.
- Fix or mitigation plan communicated as soon as it is available.

We will credit reporters who request acknowledgement once a fix is released.
